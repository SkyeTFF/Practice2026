# Octagon Book Store API

Веб-приложение для управления книжным магазином (книгами и категориями). 
Проект построен на современном стеке **Python + FastAPI** с использованием ORM **SQLAlchemy 2.0** и базы данных **PostgreSQL** внутри среды **WSL**.

---

## 🛠️ Стек технологий
- **Язык программирования:** Python 3.10+
- **Веб-фреймворк:** FastAPI (Асинхронный)
- **Валидация данных:** Pydantic v2
- **ORM (База данных):** SQLAlchemy + драйвер `psycopg2-binary`
- **СУБД:** PostgreSQL

---

## 🚀 Как запустить проект локально (в WSL)

### 1. Настройка базы данных PostgreSQL
Убедитесь, что служба PostgreSQL запущена и в ней создан необходимый пользователь:
```bash
sudo service postgresql start
```

### 2. Клонирование и настройка окружения
Разверните виртуальное окружение и установите все необходимые зависимости:
```bash
# Создание и активация окружения
python3 -m venv venv
source venv/bin/activate

# Установка библиотек
pip install -r requirements.txt
```

### 3. Настройка переменных окружения
Создайте в корне проекта файл `.env` и укажите параметры подключения к вашей локальной базе данных:
```ini
DB_HOST=localhost
DB_PORT=5432
DB_NAME=octagon
DB_USER=octagon
DB_PASSWORD=12345
```

### 4. Наполнение базы тестовыми данными (Опционально)
Вы можете запустить скрипт первоначальной инициализации, чтобы автоматически создать таблицы и добавить несколько стартовых категорий и книг:
```bash
python3 -m app.init_db
```

### 5. Запуск веб-сервера
Запустите сервер разработки Uvicorn:
```bash
uvicorn app.main:app --reload
```
После этого API станет доступно по адресу: `http://127.0.0.1:8000`
